# Anneler Günü Hediyesi 🌸

Bu proje, Anneler Günü için özel olarak hazırlanmış, etkileşimli bir web uygulamasıdır.

## Özellikler
- Şık bir zarf açılış animasyonu.
- Kalp ve çiçek efektli kutlama.
- Kişiselleştirilmiş video sürprizi.

## Nasıl Kullanılır?
1. Bu projeyi bilgisayarınıza indirin.
2. `mom_gift.mp4` adında bir video dosyasını ana dizine ekleyin.
3. `index.html` dosyasını tarayıcıda açın veya bir web sunucusunda yayınlayın.
